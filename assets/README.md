# Assets Directory

This directory contains downloadable files for the website.

## Files to Add:

1. **cv.pdf** - Your curriculum vitae in PDF format
   - Update this file with your actual CV
   - Make sure it's named exactly `cv.pdf` to work with the download links

## Adding Your CV:

1. Export your CV as a PDF
2. Name it `cv.pdf`
3. Place it in this directory (`assets/cv.pdf`)
4. The download links on the website will automatically work

## Optional Files:

You can also add:
- Research papers
- Writing samples
- Transcripts (if you want to make them available)
- Portfolio items

Just update the links in the HTML files to point to these assets.
