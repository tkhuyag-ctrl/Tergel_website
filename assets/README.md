# Assets Directory

This directory contains downloadable files and images for the website.

## Files to Add:

1. **profile-photo.jpg** - Your profile photo for the home page
   - Use a professional headshot
   - Name it exactly `profile-photo.jpg` (or .png, .jpeg)
   - Recommended size: at least 400x400 pixels (square format works best)
   - The photo will automatically display on the home page hero section

2. **cv.pdf** - Your curriculum vitae in PDF format
   - Update this file with your actual CV
   - Make sure it's named exactly `cv.pdf` to work with the download links
   - You can also upload your CV through the admin interface on the CV page

## Adding Your Profile Photo:

1. Choose a professional headshot photo
2. Name it `profile-photo.jpg` (or .png, .jpeg)
3. Place it in this directory (`assets/profile-photo.jpg`)
4. The photo will automatically appear on the home page

## Adding Your CV:

1. Export your CV as a PDF
2. Name it `cv.pdf`
3. Place it in this directory (`assets/cv.pdf`)
4. The download links on the website will automatically work

## Optional Files:

You can also add:
- Research papers
- Writing samples
- Transcripts (if you want to make them available)
- Portfolio items
- Project screenshots

Just update the links in the HTML files to point to these assets.
